## API Production Dockerfile

Security-hardened container for sovereign API deployment.

### Security Features

- Non-root user execution
- Minimal Python 3.12-slim base
- Multi-worker Uvicorn configuration

### Build Process

```dockerfile
FROM python:3.12-slim

RUN groupadd -r iluminara && useradd -r -g iluminara iluminara

WORKDIR /app
COPY requirements-ml.txt .
RUN pip install --no-cache-dir -r requirements-ml.txt

COPY . .
RUN chown -R iluminara:iluminara /app
USER iluminara

EXPOSE 8000
CMD ["uvicorn", "api.server:app", "--host", "0.0.0.0", "--port", "8000", "--workers", "4"]
```

### Production Ready

- 4 worker processes for scalability
- Health check endpoints
- Logging and monitoring integration
