---
title: 'DSPM Engine'
description: 'Data Security Posture Management for System 2.'
---

The DSPM Engine provides continuous monitoring and risk assessment for all data flows in iLuminara.

- **Discovery**: Maps all data sources and sinks
- **Classification**: Tags sensitive and regulated data
- **Monitoring**: Detects policy violations in real time
- **Remediation**: Triggers System 2 Reasoning Loop for violations

This ensures that data security posture is always aligned with the latest global standards.
