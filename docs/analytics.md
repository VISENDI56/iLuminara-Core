# Analytics Integration

## Plausible (Self-Hosted)

To enable privacy-first analytics, add the following script to your documentation or Mintlify portal:

```html
<script async defer data-domain="iluminara-core" src="https://plausible.io/js/plausible.js"></script>
```

For self-hosted analytics, see: https://plausible.io/docs/self-hosted
