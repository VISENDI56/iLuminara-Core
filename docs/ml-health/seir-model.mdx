## SEIR Epidemiological Forecasting

ISO/TR 24291:2021 compliant outbreak prediction using differential equations.

### Model Components

- **SEIRModel**: Core epidemiological simulation
- Population dynamics modeling
- Parameter estimation and forecasting

### Features

- Real-time outbreak prediction
- Multi-compartment modeling
- Public health intervention planning

### Usage

```python
from ml_health.epidemiology.seir_model import SEIRModel

model = SEIRModel(population=1000000)
t, results = model.run_forecast(days=160, beta=0.3, sigma=0.2, gamma=0.1)
peak_day = np.argmax(results[2])  # Peak infection day
```
