## Quick Start

```bash

git clone https://github.com/VISENDI56/iLuminara-Core.git
cd iLuminara-Core

# Install dependencies
pip install -r requirements.txt

# Launch Sovereign Dashboard
streamlit run sentinel_ui.py

./launch_full_demo.sh
```
